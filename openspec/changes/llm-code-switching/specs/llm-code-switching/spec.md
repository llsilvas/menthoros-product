## ADDED Requirements

### Requirement: System prompt in English
The system prompt (`system-prompt.txt`) SHALL be written entirely in English, covering agent role, safety rules, mandatory validations, output format constraints, and training principles.

#### Scenario: Agent role instruction is in English
- **WHEN** the LLM receives a chat request with the system prompt
- **THEN** the system prompt content is in English with no Portuguese instructions

### Requirement: Technical context sections in English
All structural headings, metric labels, alert markers, and analysis instruction blocks generated by `PlanoTreinoPromptBuilder` and its formatters SHALL be in English.

#### Scenario: Section headings use English labels
- **WHEN** `buildOptimizedPrompt` assembles the full prompt
- **THEN** section headings such as `## Physiological Data and Training Zones`, `## Load Metrics`, `## Recent Training History`, and `## Goals for This Week` appear in English

#### Scenario: Decision markers are in English
- **WHEN** `formatarDecisaoIntervalado` generates the interval training decision block
- **THEN** the markers are `[FORBIDDEN]`, `[DEGRADED]`, and `[AUTHORIZED]` (not `[PROIBIDO]`, `[DEGRADADO]`, `[AUTORIZADO]`)

#### Scenario: Alert markers are in English
- **WHEN** `AlertasPromptFormatter` generates mandatory alerts and decision hierarchy
- **THEN** alert headers and labels are in English

### Requirement: Athlete input preserved in Portuguese
All athlete data fields (name, goal, observations, workout feedback, notes) injected into the prompt SHALL remain in Portuguese.

#### Scenario: Athlete name and goal are in Portuguese
- **WHEN** the prompt is assembled with athlete data
- **THEN** the `Nome` / `Name` placeholder value is the athlete's name as stored, and `Objetivo` / `Goal` value uses the athlete's original objective text in Portuguese

#### Scenario: Workout observations remain in Portuguese
- **WHEN** recent training history is formatted by `formatarHistoricoTreinos`
- **THEN** observation text from `TreinoRealizado.observacao` is included as-is in Portuguese

### Requirement: JSON output keys in English, free-text values in Portuguese
The LLM output JSON SHALL use English field names (as defined by the Java record DTOs). Free-text fields (`justificativaIa`, `descricaoEtapa`, `objetivoSemanal`) SHALL be filled in Brazilian Portuguese.

#### Scenario: LLM fills justificativaIa in Portuguese
- **WHEN** the LLM generates a weekly plan
- **THEN** the `justificativaIa` field of each planned workout contains text in Brazilian Portuguese

#### Scenario: LLM fills descricaoEtapa in Portuguese
- **WHEN** the LLM generates workout stages
- **THEN** the `descricaoEtapa` field of each stage contains text in Brazilian Portuguese

### Requirement: Sports terminology preserved in English within Portuguese text
Domain-specific English terms SHALL be preserved as-is even when they appear within Portuguese sentences: `threshold`, `TSS`, `ATL`, `CTL`, `TSB`, `VO2max`, `Z1`â€“`Z5`, `HR drift`, `long run`, `tempo run`, `fartlek`, `decoupling`.

#### Scenario: English sports terms appear unchanged in Portuguese output
- **WHEN** the LLM generates a `justificativaIa` for a tempo run workout
- **THEN** the term `tempo run` and zone identifiers like `Z3` or `Z4` appear in English within the Portuguese sentence

### Requirement: Optimized template structure in English with Portuguese data placeholders
The `plano-treino-otimizado-claude.txt` template SHALL use English for all structural sections, instructions, and analysis directives. Data placeholders (`%s`, `%d`) inject Portuguese athlete data and remain unchanged.

#### Scenario: Template instructions are in English
- **WHEN** `PromptTemplateLoader` loads `plano-treino-otimizado-claude.txt`
- **THEN** the static instruction text (headings, rules, analysis steps) is in English

#### Scenario: Athlete data placeholder values are in Portuguese
- **WHEN** the template is formatted with `String.format` in `buildOptimizedPrompt`
- **THEN** the substituted values for athlete name, goal, and history text are in Portuguese
